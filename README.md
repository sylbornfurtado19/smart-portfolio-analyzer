# Smart Portfolio Analyzer

A full-stack fintech application for tracking investment portfolios, analyzing performance, assessing risk, and comparing returns against market benchmarks. Built with React, Python, MySQL, and financial data APIs to deliver real-time portfolio insights and analytics.

---

## Features

* User Authentication & Portfolio Management
* Real-Time Stock Data Integration
* Portfolio Value Tracking
* Profit & Loss Analysis
* Sector-wise Asset Allocation
* Diversification Analysis
* Risk Assessment & Volatility Tracking
* Sharpe Ratio & Beta Calculation
* Benchmark Comparison (NIFTY 50, SENSEX)
* Interactive Charts & Visualizations
* Portfolio Health Score
* AI-Powered Portfolio Insights (Upcoming)

## Core Analytics

### Portfolio Allocation

Analyze investments based on sectors and asset distribution.

### Diversification Score

Measure how diversified a portfolio is across industries.

### Risk Analysis

Evaluate portfolio volatility and overall risk exposure.

### Sharpe Ratio

Assess risk-adjusted returns.

### Beta Analysis

Compare portfolio volatility against the broader market.

### Benchmark Comparison

Compare performance with major market indices.

### Portfolio Health Score

A consolidated score based on diversification, returns, risk, and liquidity.
